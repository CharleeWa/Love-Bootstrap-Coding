# Love-Bootstrap-Coding

正确的方式编写 `Bootstrap` 网站，以及提供一个快速启动脚手架。

```
template/
  ├──  assets
  │   ├──  css/
  │   ├──  images/
  │   ├──  js/
  │   ├──  scss/
  │   └──  vendor/
  └── All template HTML files
gulpfile.js
package.JSON
package-lock.JSON
README.MD
```

Note. This project use `Bootstrap v5.2.0`, It does not support any version of IE, including IE 11.
